# Movie Info Extractor

An AI-powered application that converts unstructured movie descriptions into structured, validated data using LangChain and Mistral AI. The extracted information can be previewed or stored directly in PostgreSQL and MongoDB through an interactive Streamlit interface.

## Tech Stack
- Python
- LangChain
- Mistral AI
- Pydantic
- Streamlit
- PostgreSQL
- MongoDB
